There are two ways to prove that a greedy algorithm is optimal. They are:


**1. â€˜Greedy stays aheadâ€™ arguments :-**

Using a â€˜Greedy stays aheadâ€™ argument is one of the simplest methods to prove that a greedy algorithm is correct. It shows that according to some measure, the greedy algorithm always is at least as far ahead as the optimal solution during each iteration of the algorithm. This fact can be used to prove that the greedy algorithm is optimal.



**2. Exchange Arguments :-**


This is a rather versatile technique. It essentially shows you that it is possible for you to iteratively transform any optimal solution into the solution produced by the greedy algorithm without making any change .
